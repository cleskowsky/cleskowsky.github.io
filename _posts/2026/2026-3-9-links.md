* [Slashing agent token costs by 98% with RFC 9457-compliant error responses](https://blog.cloudflare.com/rfc-9457-agent-error-pages/):
  Cloudflare adapts error responses based on the Accept header for agentic clients. Rather than returning an html 404
  error page that is not meaningful to an agent, it provides concise feedback and next steps. Prompt injection for good?
* [Canada Needs Nationalized, Public AI](https://www.schneier.com/blog/archives/2026/03/canada-needs-nationalized-public-ai.html):
  A bunch of public funding for research into ai comes available in Canada. Is it going to go to OpenAI and other
  private, for profit companies (largely US based at this point) or is there another way we can think about investing
  that money?
* [Cloudflare outage on February 20, 2026](https://blog.cloudflare.com/cloudflare-outage-february-20-2026/): Some work intended to make a manual task safer created an incident. It sounds like an api call to remove bgp addresses had a bug that was triggered by this new process and resulted in many more routes than intended being removed. Some customers were down for more than 6 hours ... :/
* [Quick takes on Feb 20 Cloudflare outage](https://surfingcomplexity.blog/2026/02/21/quick-takes-on-feb-20-cloudflare-outage/): Summary of the above outage.
* [Nobody knows how the whole system works](https://surfingcomplexity.blog/2026/02/08/nobody-knows-how-the-whole-system-works/): Truth. Our systems are built on abstraction on top of abstraction. Everyone's mental model of how things works is different. Remember this helps keep me humble.
* [FOUR HUNDRED METERS on MARS](https://www.anthropic.com/features/claude-on-mars): In which claude code helps plot a course for mars rover perseverance over 400m of martian terrain. Lots of risk here in losing a very expensive rover doing science in a place where we can't just go and fix it.
